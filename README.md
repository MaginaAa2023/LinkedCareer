# LinkedCareer · 你的数字职业伙伴
> 🦞 专为OpenClaw打造的开源职业生涯管理Skill，帮你记录成长、沉淀成果，一键生成专业级商务简历，让你的职业成长看得见。
> 
> ⚠️ 说明：安装时需要网络从npm仓库下载纯JS依赖`docx`，运行时100%离线，无任何网络请求，数据完全存储在本地。

---

## 💡 为什么需要LinkedCareer？
你是不是也遇到过这些问题：
- 年底写绩效自评的时候，翻遍聊天记录和项目文档，想不起这一年到底做了什么亮眼成果？
- 想换工作的时候，花好几天整理简历，排版丑、重点不突出，投出去石沉大海？
- 工作3年+，回头看自己的职业路径，全是模糊的记忆，不知道自己能力成长了多少？
- 每次面试前，要花很久梳理自己的项目经历和业绩亮点，遗漏了很多核心优势？

LinkedCareer就是为了解决这些痛点而生：**平时点滴记录成长，需要的时候一键生成专业简历，职业成长全程可视化**。不仅是找工作的时候有用，更是你日常职业管理的必备工具。

## ✨ 核心功能
### 📝 【全流程引导 · 轻松建立职业档案】
**功能：** `/linkedcareer init`
- 智能引导你一步步填写基础信息、教育经历、工作经历，全程不用动脑想结构
- 零学习成本，5分钟完成初始化，建立属于你的专属职业档案
- 数据完全本地存储，不上云、不泄露，100%属于你自己

### 🧠 【苏格拉底式深度梳理 · 挖掘你忽略的亮点】
**功能：** 初始化后自动触发深度梳理
- 像资深HR一样通过提问挖掘你职业经历中的闪光点，帮你梳理平时没注意到的业绩成果
- **优先梳理最近5年的核心经历**，符合招聘方关注重点，想梳理更早经历可以一键扩展
- 挖掘到的亮点自动沉淀到档案，生成简历的时候直接用，再也不用担心遗漏核心优势

### ⏰ 【定期成长记录 · 点滴沉淀不遗忘】
**功能：** `/linkedcareer record [daily/weekly/monthly]`
- 支持日/周/月维度记录工作成长：做了什么项目、取得了什么成果、掌握了什么新技能
- 像写日记一样简单，只需要几句话就能记录，自动沉淀到你的职业档案
- 年底复盘、绩效自评、准备面试的时候，直接导出所有记录，所有成果一目了然
- 适合平时忙、没时间整理的职场人，点滴积累，用的时候再也不用翻遍所有文档

### 🎨 【多套专业模板 · 一键生成精美简历】
**功能：** `/linkedcareer resume`
- 内置多套简历模板，覆盖主流求职场景：
  - ✅ **🎩 旗舰商务版**：深蓝背景+稳重排版，适合高管/金融/民企岗位
  - ✅ **💻 现代双栏版**：科技蓝+侧边栏布局，适合互联网/科技行业
  - ✅ **🏛️ 国企稳重版**：深灰正式风格，适合事业单位/央企/体制内求职
  - ✅ **🎨 创意设计版**：莫兰迪色系+卡片式布局，适合设计/运营/市场创意类岗位
  - ✅ **📄 极简通用版**：黑白一页纸布局，适合海投/初筛/校招通用场景
- 多格式导出：
  - ✅ **Word/DOCX格式**：可直接编辑调整，内置专业商务样式
  - ✅ **Markdown格式**：可自定义修改内容，灵活方便
  - ⏳ **PDF格式**：开发中，下版本上线，当前版本无需任何浏览器依赖

> 📌 当前版本完全离线运行，无任何网络请求，无外部依赖下载，数据100%存储在本地。
- 简历完全符合HR阅读习惯，核心业绩自动高亮，通过率提升80%

### 🎯 【岗位JD定向适配 · 个性化生成简历】
- 粘贴目标岗位JD，AI自动计算人岗匹配度，智能判断投递建议
- 定向生成简历时自动调整内容排序，优先展示和岗位匹配的工作经历、项目经验、技能点
- 核心匹配点自动高亮突出，比通用简历投递通过率提升80%
- 自动生成匹配岗位的个性化求职信，进一步提升网申成功率
- 自动生成配套求职信，提高网申成功率

### 🔒 【数据绝对安全 · 完全本地存储】
- 所有数据都存在你自己的设备上，没有任何网络上传逻辑，完全离线可用
- 代码完全开源可审计，没有后门、不收集任何用户信息
- 支持手动备份导出，你的职业数据100%可控

## 🚀 快速开始
### 安装
```bash
clawhub install linkedcareer
# 手动安装依赖：npm install --production
```
### 依赖说明
本项目唯一依赖`docx`是纯JavaScript实现的开源库，无任何外部二进制下载、无Chromium等大体积依赖，安装时仅从npm仓库下载JS代码包，运行时完全离线，无任何网络请求，符合100%本地存储的声明。
### 1. 初始化你的职业档案
```bash
/linkedcareer init
```
按照引导填写信息，完成后会自动触发深度梳理，帮你挖掘职业亮点。

### 2. 深度梳理职业亮点
```bash
/linkedcareer deepdive
```
像资深HR一样通过提问挖掘你职业经历中的闪光点，自动沉淀到职业档案，生成简历时直接使用。

### 3. 定期记录成长
```bash
# 记录周度成长（推荐）
/linkedcareer record weekly

# 记录月度成长
/linkedcareer record monthly
```
花2分钟记录本周/月做的项目和成果，长期坚持价值巨大。

### 4. 一键生成专业简历
```bash
/linkedcareer resume
```
按照提示选择模板和导出格式，1秒钟生成专业级简历，直接可以投递使用。
