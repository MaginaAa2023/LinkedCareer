# LinkedCareer
## 核心功能
✅ 智能引导建立职业档案
✅ 深度梳理挖掘职业亮点
✅ 定期记录成长沉淀成果
✅ 多模板生成专业简历
✅ JD定向适配提升通过率
✅ 数据完全本地存储
## 快速使用
```
/linkedcareer init     # 初始化你的职业档案
/linkedcareer deepdive # 深度梳理职业经历亮点
/linkedcareer record   # 记录近期工作成长
/linkedcareer resume   # 一键生成专业简历
```
